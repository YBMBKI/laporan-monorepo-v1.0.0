# TXT MASTERPLAN

# MANAGEMENT LAPORAN YBMBKI

**Fullstack Monorepo — Node.js + React + Mantine**

## 1. TUJUAN SISTEM

Sistem ini dibuat untuk menjadi pusat pengelolaan laporan kegiatan, penjualan, performa anggota, perhitungan insentif, gaji, dan THR dalam satu aplikasi terintegrasi.

Fokus utamanya:

* memudahkan koordinator mengirim laporan harian / mingguan lewat file XLSX
* data importer masuk rapi ke database
* admin bisa melihat laporan penjualan dan performa anggota
* sistem bisa menghasilkan laporan formal yayasan per bulan
* sistem menghitung pendapatan anggota berdasarkan role/jabatan kegiatan
* sistem menghitung THR berdasarkan point order
* tampilannya sederhana, profesional, ringan, dan tidak membingungkan user awam

## 10.7 THR

Rule awal:

* total point order qty x Rp 1.000
* nominal bisa diatur di setting admin

### Contoh

Jika anggota punya total qty deal = 120
dan nilai THR per point = Rp 1.000

maka:
**THR = 120 x 1.000 = Rp 120.000**

### Opsi pengembangan

* THR dihitung per semester
* THR dihitung per tahun
* THR bisa diberi minimum masa aktif anggota
* THR hanya untuk status anggota aktif

# 11. FORMULA BISNIS

## 11.1 Penjualan

Hanya order dengan status `deal` yang dihitung sebagai penjualan final.

### Formula

`total_penjualan = sum(total_harga untuk status deal)`

---

## 11.2 Total deal anggota

`total_deal_anggota = jumlah order status deal yang terkait anggota`

---

## 11.3 Gaji / insentif anggota

Berdasarkan jabatan kegiatan pada order deal.

### Formula umum

`insentif = jumlah order deal x tarif jabatan kegiatan`

Contoh:

* konsultan 8 order deal x 100.000 = 800.000
* surveyor 6 order deal x 50.000 = 300.000
* pribadi 3 order deal x 150.000 = 450.000

Total = 1.550.000

---

## 11.4 THR

### Formula umum

`THR = total_qty_deal x nominal_thr_per_qty`

Contoh:
100 qty x 1.000 = 100.000
